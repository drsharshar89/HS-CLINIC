---
description: Sync local changes to GitHub repository
---

This workflow will stage all changes, create a descriptive commit message based on recent work, and push them to the main branch.

// turbo-all

1. Stage all changes:
   `git add .`

2. Check status and create a commit:
   `git commit -m "Auto-sync: Improvements and updates to HS-CLINIC"`

3. Push to GitHub:
   `git push origin main`
